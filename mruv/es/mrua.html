El Movimiento Rectilíneo Uniformemente Acelerado (MRUA) o tambien conocido como Movimiento Rectilíneo Uniforme Variado (MRUV), se encarga de estudiar el movimiento que efectua un cuerpo cuyo desplazamiento es en linea recta y que además, dispone de una aceleración contante y distinta de cero. Por otra parte, la velocidad de un movimiento rectilineo uniforme acelerado cambia a traves del tiempo, es decir, que su velocidad no es contante y esto se debe a la aceleración.



Fórmula para hallar la distancia:

 Distancia = Velocidad inicial * Tiempo + 1/2 * Aceleracion * Tiempo2
Distancia = Velocidad final * Tiempo - 1/2 * Aceleracion * Tiempo2
Distancia = (Velocidad inicial + Velocidad final) / 2 * Tiempo


Fórmulas para hallar la velocidad inicial:

 Velocidad inicial = Velocidad final - Aceleración * Tiempo
Velocidad inicial = (Velocidad final - 2 * Aceleración * Distancia)1/2


Fórmulas para hallar la velocidad final:

Velocidad final = Velocidad inicial + Aceleración * Tiempo
Velocidad final = (Velocidad inicial + 2 * Aceleración * Distancia)1/2


Fórmula para hallar el tiempo:

 Tiempo = (Velocidad final - Velocidad inicial) / Aceleración


Fórmulas para hallar la aceleración:

 Aceleración = (Velocidad final - Velocidad inicial) / Tiempo
Aceleración = (Distancia - (Velocidad inicial * Tiempo) ) / (0,5 * Tiempo)2
Aceleración = (Distancia + (Velocidad final * Tiempo) ) / (0,5 * Tiempo)2
Aceleración = (Velocidad final2 - Velocidad inicial2) / 2 * Distancia
